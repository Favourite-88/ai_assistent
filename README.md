## 📝 Описание проекта

**ИИ-ассистент для юридической компании** — это решение на базе больших языковых моделей, предназначенное для:

- 🗣️ Ведения осмысленного диалога с потенциальными и текущими клиентами
- ⚖️ Предоставления точных ответов на вопросы по законодательству и внутренней базе знаний фирмы
- 📅 Автоматизации записи на консультации и синхронизации с Google Календарём

Проект создан для повышения эффективности работы юристов, сокращения времени на рутинные запросы и улучшения клиентского опыта.

## ✨ Ключевые возможности

### 💬 Умный диалог
- Поддержка контекстного общения с сохранением истории диалога
- Распознавание намерений клиента (запрос консультации, уточнение услуг, жалоба и др.)
- Адаптивный тон общения: формальный для юридических вопросов, дружелюбный для первичного контакта

### 📚 Ответы на основе базы знаний
- Интеграция с внутренней базой знаний юридической компании
- Поиск и цитирование релевантных статей, прецедентов, шаблонов документов
- Указание источников ответов для повышения доверия и прозрачности

### 🗓️ Управление встречами
- Автоматическое определение доступных слотов для консультаций
- Запись клиента через чат с подтверждением по email/SMS
- Синхронизация с **Google Calendar** через OAuth 2.0
- Напоминания о предстоящих встречах (за 24 часа и за 1 час)

### 🔐 Безопасность и соответствие
- Шифрование персональных данных клиентов
- Соответствие требованиям 152-ФЗ и GDPR
- Логирование действий без сохранения конфиденциальной информации

## 🔌 Интеграции

### Google Calendar
```yaml
integration:
  provider: google
  auth: oauth2
  scopes:
    - https://www.googleapis.com/auth/calendar.events
    - https://www.googleapis.com/auth/calendar.readonly
  features:
    - create_event
    - check_availability
    - send_reminders
```

### База знаний (RAG)
- Документы загружаются в векторное хранилище
- При запросе пользователя:
  1. Формируется поисковый запрос
  2. Извлекаются топ-5 релевантных фрагментов
  3. Контекст передаётся в Qwen для генерации ответа с цитированием

## ⚙️ Настройка и запуск

### Предварительные требования
- Аккаунт в [Savvy](https://app.suvvy.ai/)
- API-ключ Qwen (DashScope)
- Google Cloud Project с включённым Calendar API
- (Опционально) Доступ к векторной БД

### Пошаговая настройка

1. **Настройка Savvy-ассистента**
   - Создайте нового бота в панели Savvy
   - Загрузите системный промпт (см. ниже)
   - Настройте триггеры для передачи управления в календарь

2. **Системный промпт для Qwen**
```text
Ты — профессиональный ИИ-ассистент юридической компании "Право и Партнёры".
Твоя задача:
- Отвечать на вопросы клиентов по гражданскому, семейному, трудовому праву
- Не давать юридических консультаций, а направлять к специалистам
- Вежливо запрашивать контактные данные для записи на встречу
- Использовать только информацию из предоставленной базы знаний
- Указывать источник ответа, если цитируешь документ
- При запросе записи: уточнять дату, время, тип консультации и передавать данные в интеграцию с календарём

Важно: не выдумывай законы и прецеденты. Если информации нет — честно сообщи об этом.
```

3. **Подключение Google Calendar**
   - В Google Cloud Console создайте OAuth 2.0 Client ID
   - Добавьте redirect URI из панели Savvy
   - Предоставьте доступ к календарю при первой авторизации

4. **Загрузка базы знаний**
   - Подготовьте документы в формате `.pdf`, `.docx` или `.txt`
   - Разбейте на логические блоки (статьи, услуги, FAQ)
   - Загрузите через интерфейс Savvy или API векторной БД

## 💬 Примеры использования

### Сценарий 1: Запрос по услуге
```
Клиент: Какие документы нужны для развода?

Ассистент: 
Для подачи заявления о расторжении брака в РФ потребуются:
1. Паспорта обоих супругов
2. Свидетельство о заключении брака
3. Квитанция об оплате госпошлины (650 ₽)
4. Заявление по форме №10 (заполняется на месте)

📌 Источник: Семейный кодекс РФ, ст. 19-21; внутренний регламент фирмы.

Хотите записаться на консультацию по подготовке документов? 📅
```

### Сценарий 2: Запись на встречу
```
Клиент: Да, запишите меня на завтра

Ассистент:
На завтра, 15 мая, доступны следующие слоты:
• 11:00 — 12:00
• 16:30 — 17:30

Какой вам удобнее? Укажите, пожалуйста, ваше имя и номер телефона для подтверждения.
```

### Сценарий 3: Вопрос вне базы знаний
```
Клиент: Как оспорить завещание после смерти наследодателя?

Ассистент:
Этот вопрос требует детального анализа конкретной ситуации. 
Я передам ваш запрос старшему юристу по наследственному праву.

Можете ли вы указать удобное время для звонка? 
Или запишитесь на очную консультацию: [ссылка на календарь]
```

## 🔐 Безопасность и конфиденциальность

- ✅ Все персональные данные шифруются при передаче и хранении
- ✅ Доступ к календарю — только на создание/чтение событий, без доступа к личным письмам
- ✅ Логи диалогов хранятся обезличенно, без ПДн
- ✅ Регулярный аудит доступа и соответствия 152-ФЗ
- ✅ Возможность экспорта и удаления данных по запросу клиента (право на забвение)
